## 1. 光纤光学基础

### 1.1 什么是光纤?

**定义**: 光纤(Optical Fiber)是一种能够传输光信号的细长玻璃或塑料丝,其核心结构包括:

```
┌─────────────────────────────────────┐
│        包层 (Cladding)              │  ← 外层,折射率较低
│  ┌───────────────────────────────┐  │
│  │      纤芯 (Core)              │  │  ← 内层,折射率较高
│  │   (光信号在此传输)            │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
│←─ 纤芯直径: 9μm (单模) / 50μm (多模) ─→│
│←─ 包层直径: 125μm ─────────────────────→│
```

**核心概念**:
- **纤芯 (Core)**: 光信号传输的主要通道
- **包层 (Cladding)**: 包围纤芯,折射率略低于纤芯
- **涂覆层 (Coating)**: 保护光纤,通常为聚合物

### 1.2 全内反射 - 光纤传输的物理基础

**基本原理**: 光从高折射率介质射向低折射率介质时,如果入射角大于临界角,光将完全反射回高折射率介质,不发生折射。

**数学表达**:

临界角 θc 由斯涅尔定律给出:

```
sin(θc) = n₂ / n₁
```

其中:
- n₁ = 纤芯折射率 (约 1.468)
- n₂ = 包层折射率 (约 1.463)
- θc = 临界角

**计算示例**:
```
θc = arcsin(1.463 / 1.468)
   = arcsin(0.9966)
   ≈ 85.2°
```

这意味着: 光在纤芯-包层界面的入射角必须大于 85.2°,才能发生全内反射[1]。

**为什么重要?**
- 全内反射确保光在光纤中传输数公里而不泄漏
- 这是光纤通信和传感能够实现的物理基础[1]

### 1.3 光纤的模式

**模式 (Mode)**: 光在光纤中传输的特定路径或场分布。

#### 1.3.1 单模光纤 (Single-Mode Fiber, SMF)

**特点**:
- 纤芯直径: ~9 μm
- 只允许一种模式传输
- 无模间色散,适合长距离传输
- 工作波长: 1310 nm, 1550 nm

**优势**:
- 传输距离长(可达 100 km)
- 带宽大
- 衰减低

**应用**:
- 长距离通信
- 高精度传感(如 φ-OTDR)

#### 1.3.2 多模光纤 (Multi-Mode Fiber, MMF)

**特点**:
- 纤芯直径: 50 μm 或 62.5 μm
- 允许多种模式同时传输
- 存在模间色散,限制传输距离
- 成本较低

**应用**:
- 短距离通信(局域网)
- 某些强度调制型传感

### 1.4 光纤中的衰减

**定义**: 光信号在光纤中传输时功率逐渐减小的现象。

**衰减来源**:

| 来源 | 机制 | 典型值 |
|------|------|--------|
| **瑞利散射** | 微观折射率不均匀 | ~0.12 dB/km @ 1550 nm |
| **吸收损耗** | 材料杂质(OH⁻,金属离子) | <0.01 dB/km (现代光纤) |
| **弯曲损耗** | 光纤弯曲导致模式泄漏 | 宏弯/微弯 |
| **连接损耗** | 接头、熔接点 | 0.1-0.5 dB/点 |

**Keck 的贡献**: 1970年代,Corning 公司的 Keck 等人通过材料纯化,将光纤衰减降低到 20 dB/km 以下,这是光纤通信实用化的转折点[2]。

**典型衰减值**:
- 1310 nm: ~0.35 dB/km
- 1550 nm: ~0.20 dB/km (最低衰减窗口)

## 2. 光散射原理

光散射是光纤传感的物理基础。当光在光纤中传输时,会与材料发生相互作用,产生不同类型的散射。

### 2.1 散射的分类

按散射机制的能量变化分类:

```
                        光散射
                    ↗          ↘
              弹性散射          非弹性散射
            (能量不变)          (能量变化)
                ↓                   ↓
            瑞利散射          布里渊散射  拉曼散射
                              (声子)    (分子振动)
```

### 2.2 瑞利散射 (Rayleigh Scattering)

#### 2.2.1 基本原理

**定义**: 光与比波长小得多的粒子(如材料密度波动)发生弹性散射,散射光频率与入射光频率相同。

**物理机制**:
- 光纤玻璃的微观折射率不均匀性
- 这些不均匀性来源于制造过程中的热涨落

**强度特征**:

瑞利散射强度 I_R 与波长的四次方成反比:

```
I_R ∝ 1 / λ⁴
```

这意味着: 波长越短,瑞利散射越强。

**典型强度**:
- 1550 nm: 约 -50 dB (相对入射光)
- 1310 nm: 约 -47 dB
- 850 nm: 约 -40 dB

**为什么瑞利散射最强?**
- 瑞利散射是光纤中最强的后向散射信号
- 比布里渊散射强约 20-30 dB
- 比拉曼散射强约 30-40 dB

#### 2.2.2 瑞利散射在传感中的应用

**核心技术**: φ-OTDR (相位敏感光时域反射技术)

**工作原理**:
```
1. 发射窄线宽激光脉冲进入光纤
2. 光纤各点产生后向瑞利散射
3. 接收散射光信号
4. 分析散射光的相位或强度变化
5. 提取振动/应变信息
```

**关键要求**:
- **窄线宽激光器**: 线宽 < 100 kHz,确保相干性
- **高信噪比探测**: 需要光电探测器(PD)和数据采集系统(DAQ)
- **相位解调算法**: 提取微小的相位变化

**应用场景**:
- 振动监测 (DAS/DVS)
- 地震波探测
- 周界安防

**文献示例**: Duckworth (2013) 综述了瑞利散射在分布式传感中的应用[3]

### 2.3 布里渊散射 (Brillouin Scattering)

#### 2.3.1 基本原理

**定义**: 光与光纤中的声学声子(晶格振动)发生相互作用,产生频率偏移的散射光。

**物理过程**:
```
入射光子(ν₀)  +  热声子        →  斯托克斯光子(ν₀ - νB)
入射光子(ν₀)  +  吸收声子       →  反斯托克斯光子(ν₀ + νB)
```

其中 νB 是布里渊频移。

**布里渊频移**:

在石英光纤中,布里渊频移约为:

```
νB ≈ 11 GHz (在 1550 nm 波长)
```

**频移与温度/应变的关系**:

```
ΔνB = C_T · ΔT + C_ε · Δε
```

其中:
- C_T ≈ 1.0 MHz/°C (温度系数)
- C_ε ≈ 500 MHz/% (应变系数)
- ΔT = 温度变化
- Δε = 应变变化

#### 2.3.2 布里渊散射在传感中的应用

**主要应用**: 分布式温度和应变测量

**技术类型**:

| 技术 | 测量方式 | 特点 |
|------|----------|------|
| **BOTDR** | 自发布里渊散射 | 单端测量,精度较低 |
| **BOTDA** | 受激布里渊散射 | 双端测量,精度高 |

**优势**:
- 温度和应变同时测量
- 传感距离长 (可达 50-100 km)
- 精度高 (应变分辨率 ~10 με)

**局限**:
- 响应慢 (扫描时间长)
- 系统复杂
- 成本高

**文献示例**: Tanaka (1997) 报道了基于布里渊散射的光纤温度传感器[4]

### 2.4 拉曼散射 (Raman Scattering)

#### 2.4.1 基本原理

**定义**: 光与分子振动/转动能级发生相互作用,产生频率偏移较大的散射光。

**物理过程**:
```
入射光子(ν₀)  +  分子(v=0)     →  斯托克斯光子(ν₀ - νR) + 分子(v=1)
入射光子(ν₀)  +  分子(v=1)     →  反斯托克斯光子(ν₀ + νR) + 分子(v=0)
```

其中 νR 是拉曼频移 (约 13 THz 在石英中)

**斯托克斯与反斯托克斯光**:
- **斯托克斯光 (Stokes)**: 能量降低,频率减小
- **反斯托克斯光 (Anti-Stokes)**: 能量升高,频率增大

**温度依赖性**:

反斯托克斯与斯托克斯光强度比与温度相关:

```
I_AS / I_S ∝ exp(-hνR / kT)
```

其中:
- h = 普朗克常数
- k = 玻尔兹曼常数
- T = 绝对温度

这是拉曼散射测温的基础。

#### 2.4.2 拉曼散射在传感中的应用

**主要应用**: 分布式温度传感 (DTS)

**工作原理**:
```
1. 发射激光脉冲进入光纤
2. 测量后向拉曼散射的斯托克斯和反斯托克斯光
3. 计算两者强度比
4. 根据强度比计算温度
```

**特点**:
- **只对温度敏感**: 对应变不敏感
- **精度高**: 可达 ±0.5°C
- **距离中等**: 2-8 km

**局限**:
- 信号弱 (比瑞利散射低 30 dB)
- 需要高功率激光器
- 长距离受限

**文献示例**: Farahani (1999) 详细分析了基于拉曼散射的分布式温度传感系统[5]

### 2.5 三种散射的比较

| 特性 | 瑞利散射 | 布里渊散射 | 拉曼散射 |
|------|----------|------------|----------|
| **频率偏移** | 0 | ~11 GHz | ~13 THz |
| **相对强度** | 最强 (基准) | 低 ~30 dB | 低 ~40 dB |
| **敏感量** | 应变、振动 | 温度、应变 | 温度 |
| **传感距离** | 中等 (10-50 km) | 长 (50-100 km) | 中等 (2-8 km) |
| **响应速度** | 快 (kHz) | 慢 (分钟) | 慢 (秒) |
| **主要应用** | DAS/DVS | BOTDR/BOTDA | DTS |

## 3. 信号处理基础

### 3.1 信号与系统的基本概念

#### 3.1.1 什么是信号?

**定义**: 信号是携带信息的物理量随时间变化的函数。

**分类**:

```
信号
├── 确定性信号
│   ├── 周期信号 (如正弦波)
│   └── 非周期信号 (如脉冲)
└── 随机信号
    ├── 平稳随机信号
    └── 非平稳随机信号 (如振动信号)
```

**光纤传感中的信号**:
- 后向散射光强度 → 电信号 (通过光电探测器)
- 振动信号 → 通常是非平稳随机信号

#### 3.1.2 系统的分类

**线性时不变系统 (LTI)**: 满足叠加性和时不变性。

**重要性**:
- 大部分信号处理算法基于 LTI 系统理论
- 卷积、傅里叶变换等工具适用于 LTI 系统

### 3.2 傅里叶变换 - 频域分析的基础

#### 3.2.1 基本原理

**时域与频域**:

信号可以在两个域中表示:
- **时域**: 信号幅度随时间的变化
- **频域**: 信号包含的频率成分

**傅里叶变换定义**:

连续傅里叶变换 (CFT):
```
X(f) = ∫ x(t) · e^(-j2πft) dt
```

离散傅里叶变换 (DFT):
```
X[k] = Σ x[n] · e^(-j2πkn/N)
```

**物理意义**:
- 任何时域信号都可以分解为不同频率的正弦波的叠加
- X(f) 表示频率 f 处的幅度和相位

#### 3.2.2 在光纤传感中的应用

**应用场景 1: 振动信号频谱分析**

```
时域振动信号 → FFT → 频谱
```

**用途**:
- 识别振动频率成分
- 区分不同类型的振动(如地震波 vs 噪声)
- 提取特征频率

**示例**:
- 地震波: 0.1 - 100 Hz
- 机械振动: 10 - 1000 Hz
- 声波: 100 - 10000 Hz

**应用场景 2: 滤波器设计**

通过频域分析设计滤波器:
- 低通滤波器 (去除高频噪声)
- 带通滤波器 (提取特定频率范围)
- 高通滤波器 (去除低频漂移)

**文献示例**: Wu (2014) 使用信号处理方法改善 φ-OTDR 的入侵检测性能[6]

### 3.3 数字信号处理基础

#### 3.3.1 采样定理 (Nyquist-Shannon)

**定理**: 如果信号的最高频率为 f_max,则采样频率 f_s 必须满足:

```
f_s ≥ 2 · f_max
```

**在光纤传感中的应用**:

**示例**: 如果要检测 10 kHz 的振动信号:
```
f_max = 10 kHz
f_s ≥ 20 kHz
```

实际系统通常采样 5-10 倍:
```
f_s = 50 - 100 kHz
```

**为什么重要?**
- 采样不足会导致混叠 (Aliasing)
- 混叠会导致频率信息错误

#### 3.3.2 数字滤波器

**分类**:

| 类型 | 特点 | 应用 |
|------|------|------|
| **FIR** | 有限冲激响应,线性相位 | 通用滤波 |
| **IIR** | 无限冲激响应,高效 | 实时处理 |

**常用滤波器类型**:
- **低通滤波器**: 去除高频噪声
- **高通滤波器**: 去除低频漂移
- **带通滤波器**: 提取特定频率范围
- **陷波滤波器**: 去除特定干扰频率

**在 DAS 中的应用**:
- 去除直流分量
- 抑制高频噪声
- 提取振动频带

### 3.4 时频分析

#### 3.4.1 短时傅里叶变换 (STFT)

**动机**: 傅里叶变换不能提供频率随时间变化的信息。

**STFT 定义**:
```
STFT{x(t)}(τ, f) = ∫ x(t) · w(t-τ) · e^(-j2πft) dt
```

其中 w(t) 是窗函数。

**用途**:
- 分析非平稳信号
- 观察频率随时间的变化
- 识别瞬态事件

#### 3.4.2 小波变换 (Wavelet Transform)

**基本思想**: 使用不同尺度的小波函数分析信号。

**优势**:
- 多分辨率分析
- 适合瞬态信号分析
- 更好的时频局部化

**在光纤传感中的应用**:
- 事件检测与定位
- 特征提取
- 去噪

### 3.5 相位解调算法

#### 3.5.1 为什么需要相位解调?

在 φ-OTDR 中,振动信息包含在瑞利散射光的**相位变化**中,但直接测量的是**光强**。

```
I(t) ∝ |E₁ + E₂|²
```

其中 E₁, E₂ 是不同位置散射的场。

#### 3.5.2 常用相位解调方法

**方法 1: 直接相位检测**

通过相干探测直接提取相位:
```
φ = arctan(I_Q / I_I)
```

其中 I_I, I_Q 是正交信号。

**方法 2: 差分相位检测**

计算相邻位置/时间的相位差:
```
Δφ = φ(z, t) - φ(z, t-Δt)
```

**方法 3: 3×3 耦合器法**

使用 3×3 耦合器产生相位差,解调相位信息。

**文献示例**: Hautefeuille (2021) 详细介绍了 OTDR 的原理[7]
