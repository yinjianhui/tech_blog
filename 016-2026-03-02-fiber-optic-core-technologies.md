## 1. φ-OTDR 原理与系统设计

### 1.1 什么是 φ-OTDR?

**定义**: φ-OTDR (Phase-Sensitive Optical Time Domain Reflectometry) 是一种利用瑞利散射的相位信息来检测振动和应变的光纤传感技术。

**与传统 OTDR 的区别**:

| 特性 | 传统 OTDR | φ-OTDR |
|------|-----------|--------|
| **检测信号** | 散射光强度 | 散射光相位 |
| **光源** | 宽谱光源 | 窄线宽激光器 |
| **应用** | 故障定位 | 振动/应变传感 |
| **灵敏度** | 低 | 高 (nε 级) |

**核心优势**:
- 相位信息对应变/振动高度敏感
- 可以检测纳米级的应变变化
- 单端测量,部署简便

### 1.2 工作原理

#### 1.2.1 瑞利散射的干涉模型

当光脉冲在光纤中传输时,各点产生的后向瑞利散射光会发生干涉:

```
E_total(t) = Σ E_i(t) · exp(jφ_i)
```

其中:
- E_i: 第 i 个散射点的电场
- φ_i: 该点的相位

**相位与应变的关系**:

```
Δφ = (2πn/λ) · (1 - n²/2 · [p₁₂ - ν(p₁₁ + p₁₂)]) · ΔL
```

简化为:
```
Δφ ≈ (2πn/λ) · ΔL
```

其中:
- n: 纤芯折射率 (~1.46)
- λ: 光波长 (1550 nm)
- ΔL: 光纤长度变化

**数值示例**:
```
对于 1550 nm 光波:
1 微应变 (με) = 1×10⁻⁶ 应变
ΔL = 1 m × 1×10⁻⁶ = 1 nm
Δφ = (2π × 1.46 / 1550 nm) × 1 nm
   ≈ 0.0059 rad
```

这意味着: 1 με 的应变会产生约 0.006 弧度的相位变化。

#### 1.2.2 空间分辨率

**定义**: 系统能分辨的两个事件之间的最小距离。

**决定因素**:

空间分辨率 Δz 主要由光脉冲宽度 τ 决定:

```
Δz = (c / 2n) · τ
```

其中:
- c: 光速 (3×10⁸ m/s)
- n: 纤芯折射率 (~1.46)
- τ: 脉冲宽度

**计算示例**:

| 脉冲宽度 | 空间分辨率 | 应用场景 |
|----------|------------|----------|
| 10 ns | 1 m | 高精度定位 |
| 50 ns | 5 m | 一般监测 |
| 100 ns | 10 m | 长距离传感 |

**权衡关系**:
- 脉冲越窄 → 空间分辨率越高 → 但能量越低 → 信噪比越低
- 脉冲越宽 → 空间分辨率越低 → 但能量越高 → 传感距离越长

#### 1.3.2 关键组件

**1. 窄线宽激光器**

**要求**:
- 线宽: < 100 kHz (典型值 < 10 kHz)
- 波长: 1550 nm (C 波段)
- 功率: > 10 mW
- 频率稳定性: < 1 MHz/h

**为什么需要窄线宽?**
- 确保光的相干性
- 相干长度 L_c = c / (n · Δν)
- 对于 Δν = 10 kHz: L_c ≈ 20 km

**文献**: Daiber (2014) 报道了用于相干通信的窄线宽可调谐激光器[1]

**2. 声光调制器 (AOM)**

**功能**:
- 产生光脉冲
- 调制脉冲宽度
- 控制重复频率

**参数**:
- 上升时间: < 10 ns
- 插入损耗: < 3 dB
- 消光比: > 50 dB

**3. 掺铒光纤放大器 (EDFA)**

**作用**:
- 放大光脉冲功率
- 补偿系统损耗
- 延长传感距离

**参数**:
- 增益: 20-30 dB
- 噪声系数: < 5 dB
- 输出功率: > 100 mW

**4. 光环形器**

**功能**: 实现光信号的单向传输

```
端口1 → 端口2: 脉冲光进入光纤
端口2 → 端口3: 后向散射光进入探测器
```

**参数**:
- 插入损耗: < 1 dB
- 隔离度: > 50 dB
- 方向性: > 50 dB

**5. 光电探测器 (PD)**

**类型**:
- PIN 光电二极管 (中速应用)
- 雪崩光电二极管 APD (高速应用)

**参数**:
- 带宽: > 100 MHz
- 响应度: > 0.8 A/W @ 1550 nm
- 噪声等效功率 NEP: < 10 pW/√Hz

**6. 数据采集系统 (DAQ)**

**要求**:
- 采样率: > 100 MS/s
- 分辨率: > 12 bit
- 通道数: ≥ 2 (用于正交检测)

### 1.4 性能指标

#### 1.4.1 关键参数总结

| 参数 | 典型值 | 影响因素 |
|------|--------|----------|
| **空间分辨率** | 1-10 m | 脉冲宽度 |
| **传感距离** | 10-50 km | 脉冲能量、光纤损耗 |
| **采样率** | 1-10 kHz | DAQ 带宽、处理速度 |
| **频率响应** | 0.01-10 kHz | 脉冲重复频率 |
| **应变分辨率** | 1-10 nε | 相位噪声、信噪比 |
| **定位精度** | < 5 m | 采样率、空间分辨率 |

#### 1.4.2 灵敏度与动态范围

**灵敏度**: 最小可检测应变

```
ε_min = (λ / 2πnL) · (1 / SNR)
```

**动态范围**: 最大可检测应变

```
ε_max = λ / (4πnL)
```

**示例** (L = 10 m, SNR = 60 dB):
- ε_min ≈ 1 nε
- ε_max ≈ 10 με

## 2. DAS 信号处理算法

### 2.1 信号处理流程

```
原始信号 → 预处理 → 相位解调 → 去噪 → 特征提取 → 事件识别
```

### 2.2 预处理

#### 2.2.1 背景去除

**目的**: 去除直流分量和静态背景

**方法**: 简单的减法运算

```
S_processed(z, t) = S_raw(z, t) - S_background(z)
```

其中 S_background 是无振动时的参考信号。

#### 2.2.2 归一化

**目的**: 消除光功率波动的影响

**方法**:

```
S_norm(z, t) = S_processed(z, t) / max(S_processed)
```

### 2.3 相位解调算法

#### 2.3.1 直接相位检测

**原理**: 通过 I/Q 解调直接提取相位

```
I(t) = A(t) · cos(φ(t))
Q(t) = A(t) · sin(φ(t))

φ(t) = arctan(Q(t) / I(t))
```

**优点**:
- 相位信息完整
- 灵敏度高

**缺点**:
- 需要正交解调
- 计算复杂

#### 2.3.2 差分相位检测

**原理**: 计算相邻点或相邻时间的相位差

**空间差分**:
```
Δφ(z) = φ(z) - φ(z-Δz)
```

**时间差分**:
```
Δφ(t) = φ(t) - φ(t-Δt)
```

**优点**:
- 简单易实现
- 对低频噪声不敏感

**缺点**:
- 丢失绝对相位信息
- 对量化噪声敏感

**文献**: Yang (2015) 报道了基于相位提取的分布式光纤振动传感方法[3]

### 2.4 信号去噪算法

#### 2.4.1 线性滤波器

**移动平均滤波器**:

```
y[n] = (1/M) · Σ x[n-k]
      k=0 to M-1
```

**优点**: 简单,计算快
**缺点**: 会降低频谱分辨率

**中值滤波器**:

```
y[n] = median(x[n-M/2 : n+M/2])
```

**优点**: 保留边缘,去除脉冲噪声
**缺点**: 计算量较大

#### 2.4.2 小波去噪

**原理**: 利用小波变换的多分辨率分析特性

**步骤**:
1. 选择小波基 (如 db4, sym8)
2. 分解信号到不同尺度
3. 对各层系数进行阈值处理
4. 重构信号

**阈值选择**:
- 通用阈值: λ = σ · √(2·log(N))
- Stein 无偏风险估计 (SURE)
- 启发式阈值

**文献**: Chen (2023) 报道了基于深度学习的 DAS 地震数据智能去噪方法[4]

#### 2.4.3 机器学习去噪

**传统方法**: 小波变换、卡尔曼滤波

**深度学习方法**:
- 卷积神经网络 (CNN)
- 生成对抗网络 (GAN)
- 自编码器 (AutoEncoder)

**GAN 去噪流程**:

```
噪声信号 → 生成器 → 去噪信号
                ↓
              判别器 ← 真实信号
```

**文献**: Shao (2023) 使用 GAN 对 DAS 微震数据进行去噪[5]

### 2.5 特征提取

#### 2.5.1 时域特征

**基本统计特征**:
- 均值: μ = (1/N) · Σ x[i]
- 标准差: σ = √((1/N) · Σ (x[i] - μ)²)
- 峰值: max(|x[i]|)
- 峭度: K = (1/N) · Σ ((x[i] - μ)/σ)⁴
- 偏度: S = (1/N) · Σ ((x[i] - μ)/σ)³

#### 2.5.2 频域特征

**功率谱密度 (PSD)**:

```
PSD(f) = |FFT(x(t))|²
```

**谱特征**:
- 主频: f_peak = argmax(PSD(f))
- 频带能量: E = ∫ PSD(f) df
- 谱质心: f_c = (∫ f·PSD(f) df) / (∫ PSD(f) df)

**文献**: Fontana (2025) 使用功率检测器进行多车辆轨迹估计[6]

### 2.6 事件识别与分类

#### 2.6.1 基于阈值的方法

**简单阈值**:

```
if |x(t)| > threshold:
    event_detected = True
```

**自适应阈值**:

```
threshold = μ + k · σ
```

其中 k 通常取 3-5。

#### 2.6.2 机器学习分类

**传统机器学习**:
- 支持向量机 (SVM)
- 随机森林 (Random Forest)
- K 近邻 (KNN)

**深度学习**:
- 卷积神经网络 (CNN): 适合时频图
- 循环神经网络 (RNN/LSTM): 适合时序信号
- Transformer: 注意力机制,适合长序列

**文献**: Xiong (2025) 基于神经网络和 DAS 进行声源定位[7]

## 3. 相干探测技术

### 3.1 相干探测原理

#### 3.1.1 为什么需要相干探测?

**直接探测的局限**:
- 只能检测光强度,丢失相位信息
- 对微弱信号不敏感
- 受散粒噪声限制

**相干探测的优势**:
- 同时检测幅度和相位
- 提高信噪比 (量子极限)
- 频率选择性接收

#### 3.1.2 基本原理

**相干探测**: 将信号光与本地振荡光混频,通过拍频检测信号。

```
E_signal = A_s · exp(jω_s t + jφ_s)
E_LO = A_LO · exp(jω_LO t + jφ_LO)

E_total = E_signal + E_LO
I = |E_total|²
  = |E_signal|² + |E_LO|² + 2·Re(E_signal · E_LO*)
```

其中交叉项包含信号信息。

### 3.2 外差探测

#### 3.2.1 工作原理

**频移**: 本地振荡光频率与信号光不同:

```
ω_LO = ω_s + ω_IF
```

其中 ω_IF 是中频 (IF), 通常几十 MHz 到几 GHz。

**输出电流**:

```
i_IF(t) ∝ A_s · A_LO · cos(ω_IF t + φ_s - φ_LO)
```

**特点**:
- 信号在 IF 频段,便于电子处理
- 保留相位信息
- 需要宽带探测器

**文献**: Tsukamoto (2006) 报道了使用零差相干探测的光 8 相移键号信号[8]

### 3.3 零差探测

#### 3.3.1 工作原理

**频率相同**:

```
ω_LO = ω_s
```

**输出电流**:

```
i_BB(t) ∝ A_s · A_LO · cos(φ_s - φ_LO)
```

**特点**:
- 基带信号,带宽要求低
- 灵敏度最高 (量子极限)
- 需要相位锁定

**实现方式**:
- Costas 环
- PLL (Phase Locked Loop)

**文献**: Ip (2022) 报道了使用自零差探测的数据中心光互连 DAS 系统[9]

### 3.4 相干探测系统设计

#### 3.4.1 系统框图

```
┌──────────┐     ┌──────────┐
│ 信号光源 │     │ 本地振荡 │
│(激光器)  │     │(激光器)  │
└─────┬────┘     └─────┬────┘
      │                │
      │                │
      ▼                ▼

     │                │

┌──────────────────────────┐
│       90° 光混频器        │
│   ┌──────────────────┐   │
│   │  2×2 耦合器阵列    │   │
│   └──────────────────┘   │
└────┬────────┬────────┬───┘
     │        │        │
     ▼        ▼        ▼
┌────────┐ ┌────────┐ ┌────────┐
```

#### 3.4.2 关键组件

**1. 90° 光混频器**

**功能**: 产生 I/Q 两路正交信号

**输出**:
- I 通道: A_s·A_LO·cos(φ_s - φ_LO)
- Q 通道: A_s·A_LO·sin(φ_s - φ_LO)

**2. 平衡探测器**

**优势**:
- 抑制共模噪声
- 提高信噪比 3 dB
- 消除直流分量

**3. 本地振荡激光器**

**要求**:
- 窄线宽 (< 1 kHz)
- 高功率 (> 10 mW)
- 频率可调

### 3.5 性能指标

#### 3.5.1 信噪比分析

**散粒噪声极限**:

```
SNR_quantum = (η·P_signal) / (hν·Δf)
```

其中:
- η: 量子效率
- P_signal: 信号功率
- hν: 光子能量
- Δf: 带宽

**典型值**:
- 直接探测: SNR ~ 20 dB
- 外差探测: SNR ~ 25 dB
- 零差探测: SNR ~ 30 dB

## 4. 系统性能优化

### 4.1 提高信噪比

#### 4.1.1 信号平均

**原理**: 多次测量平均可以降低噪声

```
SNR_improved = SNR_single · √N
```

其中 N 是平均次数。

**权衡**: 平均次数增加 → 响应时间变长

#### 4.1.2 相干平均

**原理**: 利用信号的相位信息进行平均

**优势**: 比简单平均效果更好

**文献**: Torkamani (2021) 提出了基于模型的分布式压缩感知贝叶斯算法[10]

### 4.2 扩展传感距离

#### 4.2.1 光放大

**EDFA 放大**:
- 补偿光纤损耗
- 典型增益: 20-30 dB
- 注意: 引入放大器噪声

**拉曼放大**:
- 分布式放大
- 噪声系数更低
- 系统更复杂

#### 4.2.2 降低损耗

**方法**:
- 使用低损耗光纤 (0.16 dB/km @ 1550 nm)
- 减少熔接点
- 优化连接器

### 4.3 提高空间分辨率

#### 4.3.1 压缩感知

**原理**: 利用信号的稀疏性,突破奈奎斯特采样限制

**优势**:
- 减少数据量
- 提高分辨率

**文献**: Chen (2024) 提出了 SigRecover 方法从噪声中恢复 DAS 信号[11]
