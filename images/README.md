# Images

此文件夹用于存储技术博客中使用的图片资源。

## 📸 图片来源

- AI 助手对话中发送的图片
- 博客文章的配图
- 截图、图表、示意图

## 📁 命名规范

建议使用有意义的文件名：
- `post-title-date-01.png`
- `diagram-concept.svg`
- `screenshot-feature-001.jpg`

## 🔗 引用方式

在 Markdown 博客中引用图片：

```markdown
![图片描述](./images/filename.png)
```

或者使用相对路径：

```markdown
![图片说明](/images/filename.png)
```
